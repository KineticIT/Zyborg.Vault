Update-VltAuthToken

NAME

Update-VltAuthToken

SYNOPSIS

Renew an auth token, extending the amount of time it can be used.

Comparable to the vault CLI 'token-renew' command.

SYNTAX

Update-AuthToken [-Token] <string> [-Increment <int>] [-VaultAddress <string>] [-VaultProfile <string>] [-VaultSession <object>] [-VaultToken <string>] [<CommonParameters>]

Update-AuthToken -Self <SwitchParameter> [-Increment <int>] [-VaultAddress <string>] [-VaultProfile <string>] [-VaultSession <object>] [-VaultToken <string>] [<CommonParameters>]

DESCRIPTION

You can either specify a specific token ID or indicate to target auth token of the client caller.

This command is similar to the command 'Update-VltSecretLease' which is used for renewing secret leases.

An optional increment can be given to request a certain number of seconds to increment the lease. This request is advisory; Vault may not adhere to it at all.
